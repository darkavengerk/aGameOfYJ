게임 개요

  • 영조 연간의 조선 조정을 배경으로 한 정치 전략 게임
  • 의정부·육조·삼사 등 조선 관료 체계를 직접 조종
  • 탕평책, 균역법 등 역사적 정책 카드를 활용해 세력 확장
  • 노론·소론 두 붕당 간의 갑자 카드 시스템 (60갑자 기반)

빌드 방법

# 의존성 설치
pip install Pillow

# 전체 빌드 (카드 이미지 + TTS JSON + 미리보기 HTML)
python build.py

# 강제 전체 재빌드
python build.py --force

구조

  • data/ — 카드·토큰·보드 데이터 (JSON)
  • prototype/scripts/ — 이미지 생성 스크립트
  • build.py — 통합 빌드 오케스트레이터
  • deploy/ — 빌드 결과물 (gitignore, gh-pages 브랜치로 배포)
